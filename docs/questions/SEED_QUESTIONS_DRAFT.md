## 3. 三道代表题的实现说明

### ch03-q002：修复闰年判断

目标：修复函数，使其满足“能被 400 整除，或能被 4 整除但不能被 100 整除”时返回 `True`。

待修代码：

```python
def is_leap_year(year):
    return year % 4 == 0 and year % 100 == 0 or year % 400 != 0
```

要求：保留函数名并返回布尔值，不用打印替代返回值。可见测试使用 2024；隐藏测试至少包含 2023、1900 和 2000，并同时拦截“只判断 `% 4 == 0`”等典型错误。提示按“400 的倍数 -> 100 的倍数 -> 用 `or` 组合”递进。

### ch06-q002：去重并保持顺序

目标：读取一行空格分隔的整数，按首次出现顺序输出去重结果。例如输入 `3 1 3 2 1`，输出 `3 1 2`。

题面必须明确输入至少含一个整数，参考解法使用结果列表和成员判断，不能依赖集合顺序。测试覆盖单项、全重复、交错重复和负数；使用 `lineTrimmed`。提示按“准备结果列表 -> 遍历并判断是否已存在 -> 转成字符串后 join”递进。

### ch14-q001：成绩单摘要

目标：把逗号分隔的非空成绩文本解析为整数列表，通过函数计算人数、最高分、最低分、平均分和及格人数（分数大于等于 60），按固定格式输出。

题面约束成绩范围为 0-100，阶段内不要求处理非法文本；不得把非法输入藏进隐藏测试。测试覆盖 0、100、59/60、普通值和非整数平均值。平均值固定保留两位小数，因此使用 `exact` 或 `lineTrimmed`，不能再用 numeric 容差放宽格式。建议用 harness 调用固定函数接口，避免 stdin 缺失造成无关的 `EOFError`。
